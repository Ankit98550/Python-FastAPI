# Python-FastAPI
first to create Virtual Environment in Python using command
Python -m venv env
after create Virtual Environment activate your virtual environment by using this line in CMD Terminal
scripts/activate
install FastAPI framework in Virtual Environment pip install "fastapi[standard]", "Uvicorn[standard]"
copy this code 
run this command uvicorn main:app --reload
here main is your python file name and uvicorn is to startup your server.
you can see output at this url http://127.0.0.1:8000/
